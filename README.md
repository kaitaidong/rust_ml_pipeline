installation

`$ curl https://sh.rustup.rs -sSf | sh`

Alternatively, add the following line to your `~/.bash_profile`:

`$ export PATH="$HOME/.cargo/bin:$PATH"`

check if Rust has been installed, run 

`$ rustc --version`

Create a new Rust project

`$ mkdir ~/projects`

`$ cd ~/projects`

`$ mkdir hello_world`

`$ cd hello_world`
